üìä How to Import Your Trading Data

Charles Schwab

  1. Log into your Schwab account
  2. Go to Accounts ‚Üí History & Statements
  3. Select Transactions
  4. Choose your date range (recommend full year)
  5. Click Export ‚Üí CSV format
  6. Save the file and use "Import CSV" button above

Robinhood

  1. Open Robinhood app or website
  2. Go to Account ‚Üí Statements & History
  3. Select Account Statements
  4. Download your monthly statements
  5. Or use Tax Documents for 1099 data
  6. Export as CSV and import here

E*TRADE

  1. Log into E*TRADE Power or website
  2. Go to Accounts ‚Üí Complete View
  3. Click Transactions tab
  4. Set your date range
  5. Click Export ‚Üí CSV
  6. Import the downloaded file

TD Ameritrade / Schwab

  1. Log into thinkorswim or TD website
  2. Go to My Account ‚Üí History & Statements
  3. Select Transactions
  4. Choose Date Range
  5. Click Export to CSV
  6. Upload the file using import feature

Fidelity

  1. Log into Fidelity.com
  2. Go to Accounts ‚Üí Activity & Orders
  3. Select History
  4. Filter for All Transactions
  5. Click Download ‚Üí Spreadsheet
  6. Save as CSV and import

Optional Columns: Fees, Commission, Description

  • Make sure CSV has column headers in first row
  • Remove any summary rows or extra formatting
  • Stock transactions only (no options, bonds, etc.)
